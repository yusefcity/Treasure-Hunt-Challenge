# Treasure-Hunt-Challenge
Treasure Hunt Challenge, a lightweight C++ console game created to demonstrate clean program organization and readable game logic. The project is intentionally compact so that new C++ developers can understand every part of the implementation without relying on external libraries or complicated frameworks.

Although the application is a simple game, its structure follows ideas commonly used in larger software projects. Different sections of the program behave like logical endpoints, separating player input, game rules, scoring, and result handling into independent steps. This approach keeps the code easy to extend and maintain.

At the end of each play session, the game asks the player to approve the final score before it is recorded. This additional confirmation step illustrates how validation stages can be integrated into interactive software while keeping the implementation straightforward and beginner-friendly.

The project is intended for educational purposes and encourages experimentation. Developers can modify the guessing rules, introduce additional levels, or expand the scoring system while preserving the simple architecture. Since only the C++ standard library is used, the game can be compiled with virtually any modern C++ compiler.
